# 剪刀石頭布小遊戲 ✂️🪨📄

這是一個使用 Python 撰寫的簡單互動小遊戲，玩家與電腦進行三回合的剪刀石頭布比賽，結束後統計勝負結果，並可選擇是否再玩一次。

---

## 🎮 遊戲規則

- 遊戲共進行三回合
- 玩家每回合輸入一個數字：
  - 1：剪刀 ✂️
  - 2：石頭 🪨
  - 3：布 📄
- 電腦會隨機出招
- 每回合會即時顯示勝負，最後統計整體結果

---

## ▶️ 執行方式

1. 確保你的電腦安裝有 Python 3（可到 [python.org](https://www.python.org/) 下載）
2. 下載此倉庫中的檔案 `rock_paper_scissors.py`
3. 在終端機或命令列中執行：

```bash
python rock_paper_scissors.py
